Rules downloader

UC Berkeley, School of Law

This rule downloader is designed to work with the formatted csv file which can be created by the scrapper

NOTE: The SEC has renewed their website. This rule downloader here won’t work, and the code here is archived.

Download

Dependency: Beautifulsoup4 (4.11.1 recommended), Pandas,requests, pdftotext.
Download: Proposed rule downloader, Final rule downloader

Description

  • The csv file should be placed at the same folder with the downloader

  • To start downloader, fill “Your csv file location” by your csv file name. 
    #This is example for proposed rule downloader
df = pd.read_csv('1995-2012 formatted.csv') #Type csv file location here
df.drop_duplicates(subset=['Proposed Rule ID'], inplace=True)

    #This is example for final rule downloader
data = pd.read_csv("1995-2012 formatted.csv") #Change this to the csv file location here

  • If you are using your custom csv file with different column name from the csv file of this project, change data["Final Rule Text"] and df["Proposed Rule Text"] into your custom csv’s column name which is storing urls for the rules.

  • If you are using your custom csv file with different column name from the csv file of this project, change string of data["Final Rule ID"] and df["Proposed Rule ID"] into your custom csv’s column name which is storing urls for the rules.

  • PDF downloader requires making folders named “rule txts” and “rule pdfs” in the same directory with the downloader script. If you want to use different directory names other than “rule txts” and “rule pdfs”, change path of the indicated parts in the downloder(It’s marked by comment in downloader).
