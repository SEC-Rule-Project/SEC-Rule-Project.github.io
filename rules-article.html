What is File number?

  1. File number is sequence of letters and numbers for each propose and final rule. You can assume if a propose rule and final rule have the same file number, then the proposed rule has been approved so that it became the final rule. In other words, file number allows to keep track of rule-making process by indicating which propose rule became final rule. Also, it can be used to check whether a proposed rule is finalized or not since there will be no corresponding file number among final rules. ↩
